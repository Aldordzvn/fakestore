Sobre El Proyecto

Este proyecto fue desarrollado para mi portafolio profesional.

Es un Ecommerce simulado que consume datos de una API externa llamada FakeStore. Tiene una estructuración de componentes dentro del framework Angular V19. Uso de HTML y SCSS para construir una interfaz responsiva cuidando los detalles visuales y de experiencia para el usuario.

El objetivo de este desarrollo fue practicar y demostrar habilidades técnicas como:

Consumo de APIs REST

Uso de servicios HttpClient para consumir o filtrar datos desde los endpoints proporcionados por la API FakeStore.

Manejo de datos

Servicios
Centralización de las llamadas HTTP en servicios para separar la lógica del componente.
Suscripciones (subscribe)
Uso de Observable.subscribe() para obtener y usar los datos en componentes, proporcionados desde la API.
Uso de filtros
Productos relacionados
Excluir productos que contengan el mismo ID
Limite de productos mostrados en secciones
Persistencia en LocalStorage
Información del carrito guardada para conservar los datos aunque el usuario cambie de ruta o recargue la página.

Diseño de interface

Simplicidad visual
El diseño es ligero, ordenado y fácil de entender.
Espaciado
Uso adecuado de márgenes y padding para que los elementos puedan ser distinguidos.
Tipografia Clara
Uso de fuentes legibles con tamaños adecuados y jerarquía visual como títulos grandes y textos descriptivos más pequeños.
Paleta de colores limitada
Fondos neutros para destacar el contenido
Colores principales con buen contraste
Interacción intuitiva
Botones definidos, iconos claros y navegación sencilla.
